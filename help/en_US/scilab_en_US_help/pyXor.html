pyXor

XORs two variables.

Syntax

pyXor(arg0, arg1)

Arguments

arg0

Variable 1 to be XORed

arg1

Variable 2 to be XORed

Description

Performs XOR of the two arguments and returns the result. Arguments can be of type int, double, string, bool or python variable.

Example

py("a = 5")
var = pyGet("a")
pyXor(var,3)
