pyGet

Retrieves variable defined through py() or pyExec().

Syntax

pyGet(string Var_Name)

Arguments

file_location

Variable name as a string.

Description

Retrieves a named variable previously defined through py() or pyExec().

Example

example.py code:

a = 5;
print(a);
--> pyExec("D:\example.py")
ans  =

"5
"

--> pyGet("a")
ans  =

5
