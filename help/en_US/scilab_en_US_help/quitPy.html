quitPy

Stops the python interpreter.

Syntax

quitPy

Description

Stops the python interpreter. Once called, startPy must again be called before using any other functions offered by SciPy.

Example

--> startPy

--> py("a=5;print(a);")
ans  =

"5
"

--> quitPy
