pyAdd

Adds two variables.

Syntax

pyAdd(arg0, arg1)

Arguments

arg0

Variable 1 to be added

arg1

Variable 2 to be added

Description

Adds the two arguments and returns the result; equivalent to arg1 + arg2. Arguments can be of type int, double, string, bool or python variable.

Example

py("a = 5")
var = pyGet("a")
pyAdd(var,3)
